# security-lab

セキュリティ学習（バグハント・CTF・書籍メモ）の蓄積リポジトリ。

## 目的

- **バグハントが出来るようになる** — 実際の脆弱性を発見できるレベルへの到達
- **CTF に定期的に取り組む** — シリーズ化してブログ記事化 ([iskwyuki/portfolio#24](https://github.com/iskwyuki/portfolio/issues/24))
- **学習過程をオープンにする** — 採用面でのアピールと、同じ道を進む人への参考資料化

## ディレクトリ索引

| ディレクトリ | 内容 |
|-------------|------|
| [`ctf/`](ctf/) | CTF write-up (picoCTF / HackTheBox / TryHackMe) |
| [`books/`](books/) | セキュリティ関連書籍の学習メモ |
| [`notes/`](notes/) | トピック別メモ (XSS / SQLi / Buffer Overflow 等) |
| [`tools/`](tools/) | 自作スクリプト・検証コード |
| [`resources/`](resources/) | 参考リンク集・学習ロードマップ |

- 機密情報（API キー・トークン・本番環境の情報）はコミットしない
