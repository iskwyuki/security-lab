# CTF

CTF (Capture The Flag) の write-up を蓄積する場所。

## 構成

| サブディレクトリ | 用途 |
|------------------|------|
| `picoctf/` | picoCTF の write-up |
| `hackthebox/` | Hack The Box の write-up |
| `tryhackme/` | TryHackMe の write-up |

## 運用

- 1 問題 1 Markdown ファイル（例: `picoctf/2024/easy1st-crypto.md`）
- 解けた問題だけでなく、解けなかった問題の「どこで詰まったか」も記録する
- 大きな取り組みは、ポートフォリオ側でブログ記事化する（シリーズ: `iskwyuki/portfolio` #24）
