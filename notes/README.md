# Notes

トピック別の技術メモ。書籍・CTF・外部記事から得た知見を横断的にまとめる。

## 想定トピック

- XSS / CSRF / Clickjacking
- SQL Injection / Command Injection
- Buffer Overflow / ROP
- 認証・認可（JWT / OAuth / セッション管理）
- 暗号（対称・公開鍵・ハッシュ）
- Web プロトコル・TLS
- リバースエンジニアリング

## 運用

- 1 トピック 1 Markdown（例: `notes/xss.md`）
- 攻撃手法 / 検出 / 対策 の 3 視点でまとめる
- 関連する CTF 問題や書籍メモへのリンクを張る
